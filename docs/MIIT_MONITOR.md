# 工信部自动监控说明

## 官方来源

当前脚本监控：

- 工业和信息化部装备工业发展中心：公告发布
- 工业和信息化部装备工业发展中心：车辆准入许可

这些页面会出现《道路机动车辆生产企业及产品》批次公告、公示、目录等内容。

## 脚本做什么

```bash
python scripts/monitor_miit.py
```

会：

1. 抓取官方页面
2. 根据关键词筛选
3. 和本地 data/state/miit_seen.json 对比
4. 只输出新增项目
5. 保存报告到 outputs/reports/

## 关键词在哪里改

```text
config/settings.json
```

字段：

```json
"keywords": ["宝马", "BMW", "i3", "iX3"]
```

## 当前边界

官方动态查询系统可能依赖 JavaScript。
第一版不会强行破解接口，只做公开页面增量监控。
这更稳，也更安全。

后续升级：
Playwright自动打开“产品分批查询”，导出结果，再进入内容生成。
